# MiCaja Backend

API REST + WhatsApp Webhook para MiCaja.

## Variables de entorno (configurar en Railway):
- `SUPABASE_URL` — URL del proyecto Supabase
- `SUPABASE_KEY` — Service role key de Supabase
- `WA_TOKEN` — Token de acceso de WhatsApp API
- `WA_PHONE_ID` — Phone number ID de WhatsApp
- `WA_VERIFY_TOKEN` — Token de verificación del webhook (default: micaja_verify_2026)
